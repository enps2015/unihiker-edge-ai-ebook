Obrigado por usar e contribuir com este projeto.

## Onde tirar duvidas

Use GitHub Discussions para perguntas tecnicas, editoriais e de uso do conteudo.

- Categoria sugerida: Q&A
- Antes de abrir uma pergunta, pesquise se ja existe topico semelhante.

## Onde reportar problemas

Use GitHub Issues para bugs e problemas objetivos.

Inclua:
- contexto,
- passos para reproduzir,
- resultado esperado,
- resultado observado,
- arquivos impactados.

## Como solicitar melhorias

Abra uma issue do tipo Feature Request com:
- problema que deseja resolver,
- proposta de solucao,
- impacto esperado,
- alternativas avaliadas.

Para melhorias de conteudo, detalhe o capitulo/secao e justificativa tecnica.

## Como entrar em contato

- Discusses publicas: GitHub Discussions
- Defeitos e tarefas: GitHub Issues
